# LoggerKit 性能基准测试

本目录包含LoggerKit的性能基准测试。

## 运行基准测试

```bash
dart benchmark/logger_kit_benchmark.dart
```

## 测试项目

### 1. 基础日志性能测试
测试基本的日志记录性能，不包含任何输出。

### 2. 不同日志级别性能测试
测试各个日志级别（Debug、Info、Warning、Error、Fatal）的性能差异。

### 3. 带数据的日志性能测试
测试携带额外数据（Map）的日志记录性能。

### 4. 文件写入性能测试
测试日志写入文件的性能。

### 5. 并发日志性能测试
测试多线程并发写入日志的性能。

## 性能指标

每个测试会输出以下指标：
- **Iterations**: 迭代次数
- **Total Time**: 总耗时（毫秒）
- **Average**: 平均每次操作耗时（毫秒）
- **Throughput**: 吞吐量（操作/秒）

## 优化建议

根据基准测试结果，可以考虑以下优化方向：
1. 如果文件写入性能较低，考虑使用批量写入或异步队列
2. 如果并发性能不佳，考虑使用锁或消息队列优化
3. 如果格式化耗时较长，考虑延迟格式化或缓存格式化结果
